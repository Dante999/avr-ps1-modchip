## Pinout
this is the pinout on the playstation 1 mainboard:

    C325 Minus      GND
    C325 Plus       3.3V

    IC304 Pin24     SUBQ
    IC304 Pin26     SQCK

    IC732 Pin5      WFCK
    IC732 Pin42     CEO
